<h2>소개</h2>
AI로 문제를 만들고 풀어볼 수 있는 서비스 입니다.
<h3>특징</h3>
- 작성자가 직접 문제 내용을 적을 필요 없이 주제만 입력하면 AI가 문제를 만들어줍니다.<br>
- 만들어진 문제는 사이트에 공유되며 다른 사용자들 또한 풀어볼 수 있습니다.<br>
- 문제별로 점수가 매겨지며 점수는 랭킹 형태로 게시됩니다.
